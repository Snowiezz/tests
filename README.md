# Top-Down Shooter Game

A simple top-down shooter game built with Pygame.

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## How to Play

Run the game:

```bash
python testing.py
```

### Controls

- **WASD** or **Arrow Keys**: Move the player
- **Left Click**: Shoot towards mouse cursor
- **R**: Restart game (when game over)

### Gameplay

- You control the green circle (player)
- Red circles are enemies that spawn from the edges and chase you
- Shoot yellow bullets to destroy enemies
- Each enemy destroyed gives you 10 points
- Enemies damage you on contact
- Game ends when your health reaches 0
- Try to survive as long as possible and get the highest score!

## Features

- Player movement with WASD or arrow keys
- Mouse-based shooting mechanics
- Enemy spawning system
- Collision detection
- Health system
- Score tracking
- Game over and restart functionality
